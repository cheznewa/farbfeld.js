# Farbfeld.js
This Javascript Program, Working Render A Farbfeld Image In HTML
## How To Use This Program?
You Insert The `farbfeld.js` And Insert Canvas Images Method In `render_farbfeld_coloreffectreplacer(varimg,"add","canvas",[80,20,0])`.
## How To Insert A Image Into Javascript?
You Can Use [Base256](https://github.com/lvandeve/base256) For Insert A Image Into Javascript With The Arg `--format=js`.
## Render With Color Effect Replacer
A Full Effects From [ColorEffectReplacer](https://github.com/cheznewa/MyGist/tree/master/coloreffectreplacer) With The Function `render_farbfeld_coloreffectreplacer(varimg,"add","canvas",[80,20,0])` For Example.
## Render With FF2Clear
A Full Clearings (And Back) From [FF2Clear](https://github.com/cheznewa/MyGist/blob/master/ff2clear.py) With The Function `render_farbfeld_ff2clear(varimg,"rgb","canvas")` (And `render_farbfeld_clear2ff(varimg,"rc","canvas")`) For Example.
